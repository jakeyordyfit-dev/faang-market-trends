# FAANG Market Trends Pipeline

## Overview
Financial data pipeline analyzing 23 years of FAANG stock performance (1997-2020). Merges five separate datasets using pd.concat() and engineers a daily price volatility metric.

## Tools Used
- Python (pandas)
- Tableau Public

## Analysis
- Average annual closing price by company (1997-2020)
- Average daily price volatility by company
- Comparative growth trends across Amazon, Apple, Facebook, Google, and Netflix
